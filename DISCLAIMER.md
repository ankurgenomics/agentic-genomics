# Disclaimer

**`agentic-genomics` is a research and educational project. It is NOT a clinical tool.**

## What this means

- **Not for clinical use.** Outputs must not be used for diagnosis, treatment, prognosis, or any other clinical decision.
- **Not a medical device.** This software has not been evaluated or approved by any regulatory body (FDA, EMA, HSA, etc.) and is not intended for use as an *in vitro* diagnostic (IVD), software as a medical device (SaMD), or any other regulated category.
- **Outputs may be wrong.** Large language models can hallucinate, miscite evidence, or misapply clinical criteria (e.g., ACMG rules). All outputs must be independently verified by a qualified professional before any use beyond exploration.
- **No PHI.** Do not feed real patient data, identifiable sequencing data, or any protected health information into this tool or any third-party LLM API via this tool. Use only public, de-identified, or synthetic data.
- **Third-party APIs.** This project queries public bioinformatics resources (Ensembl, MyVariant.info, NCBI ClinVar, HPO, etc.). You are responsible for complying with their terms of service and rate limits.

## What this project *is* for

- Exploring how agentic AI patterns (tool use, planning, reasoning traces) can support — never replace — human genomic analysis.
- Serving as a reference implementation for researchers and engineers building agentic systems in bioinformatics.
- Teaching and demonstration.

## Citing this work

If you use `agentic-genomics` in research, please cite the repository. A formal citation / preprint may be added later.
